# Chronological-Client-Record-Inheritance-Case-at-Tokyo-Shinjuku-Law-Office
相続案件で東京新宿法律事務所 横浜支店に依頼しました。
以下は、私の体験に基づく事実経過と感想です。

⭐ 依頼人の意向より弁護士自身の方針が優先され、事実上の依頼放棄と感じました

相続案件で依頼しました。

しかし最終的に、
依頼人である私の意向より、弁護士本人の判断が優先される状態になり、
業務遂行の意思があるとは思えない状況に至りました。

【事実として起きたこと】
	•	相手方は内容証明すら受け取らず、協議に応じる意思がない状態
	•	私は早期の調停移行を希望
	•	しかし弁護士は「任意交渉の余地がある」と繰り返し、調停に進めない
	•	依頼者の方針より“弁護士自身の方針”が優先され続けた
	•	さらに「意見が合わないなら他の弁護士へ切り替えてください」と言われ、
　実質的に依頼を放棄された形

【特に疑問を感じた点】
	•	「依頼者が調停を望んでいるのに、なぜ進めないのか」という質問に回答なし
	•	調査すると言っていた登記調査も報告ゼロ（実施の有無すら不明）
	•	文書による進捗説明を求めても「書面回答はしない」と拒否
	•	最後は「他の弁護士に相談してください」と言われる始末

依頼契約を結んでいるのは私であり、
目的と方針を決めるのも依頼者であるはずです。

しかし実際の対応は、
弁護士の意向が最優先で、依頼者の意向は反映されませんでした。

依頼人の意思が尊重されない以上、
本来の“代理人”としての役割を果たしているとは思えません。

同じように
「依頼者の方針に沿って迅速に動いてほしい」
と思う人には、慎重な検討をおすすめします。

（※これは私自身の体験に基づくレビューです。他の担当者・案件では違う可能性もあります。）

	•	その後も、私の把握している限りでは、

　「削除しないと信頼関係の維持が難しい」といった趣旨の連絡を受け、
　私はこれを一定の圧力のように感じました。

【私の受け止め】
	•	相手方が内容証明の受取自体を拒否している段階で、
　なお任意交渉を前提とした対応が続き、
　調停・訴訟に進む具体的な動きが見えなかったことには、大きな不安と疑問を感じました。
	•	登記や財産調査についても、
　「調査を行う」との説明はあったものの、
　実施状況や結果についての説明がなく、
　着手金に見合う業務がどこまで行われているのか、
　依頼者として把握しきれませんでした。
	•	文書での説明を求めても「LINEで十分」とされ、
　進行管理や方針の共有という点で、
　私が期待していた水準とはギャップがあったと感じています。

本件は高額な相続案件であり、依頼者としては、
	•	早期の解決に向けた明確な方針提示
	•	具体的なスケジュール
	•	実際に行われた業務内容の説明

を期待していましたが、残念ながら十分には満たされていないと感じました。

そのため、現在は弁護士会への相談や、
委任解除・返金請求、新しい事務所への依頼も検討しています。

丁寧さを重視し、時間をかけて任意交渉を続けたい方には
合うケースもあるかもしれませんが、
相手方が連絡に応じない案件で、
早期に調停・訴訟へ進めたい方には、
事前に対応方針をよく確認したうえで慎重に検討されることをおすすめします。
